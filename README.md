# Clockkkkk

### Description

A simple clock app. It's annoying to use those with ads :)

### Author

Sothis-baka

### Built with

- React.js

### Prerequisites

- npm

```
npm install npm@latest -g
```

### Installation

```
npm i
npm start
```

Then your browser will pop up a new window at localhost:3000 with the app.
